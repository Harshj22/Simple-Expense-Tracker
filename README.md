# Simple-Expense-Tracker

The Expense Tracker is a beginner-level Python project that helps users manage their daily expenses easily.
Users can add, view, and delete their expenses along with category and amount details.
This project runs in the console and helps you understand Python basics like variables, lists, loops, conditions, and functions.

Features:

1. Add a new expense (with description, category, and amount)

2. View all recorded expenses

3. Calculate total expenses

4. Delete a specific expense

5. Stores expenses in a Python list (in-memory storage)


Concepts Used:

Variables & Data Types

Lists and Dictionaries

If-Else Conditions

Loops (For / While)

Basic Functions

Data Organization Logic
